How the parameters work:
  • Start Bit (S): Number of bits to skip at the beginning of the carrier file
  • Periodicity (L): Every Lth bit will be modified to store the secret message
  • Mode (C):
    • Constant: L remains the same throughout
    • Alternating: L alternates between L and 2L
    • Increasing: L increases by 4 each time, resets after L+20
    • Fibonacci: L follows the Fibonacci sequence
